# Quality Bar

## Standards
- **Performance**: < 2s app launch on low-end devices. Smooth scrolling (60fps).
- **Offline First**: Core features (viewing schedule, history) work offline.
- **Accessibility**: High contrast text, large touch targets.
- **Error Handling**: Graceful degradation, friendly error messages.

## Testing
- **Unit Tests**: Business logic coverage.
- **Widget Tests**: Critical UI component behavior.
- **Integration Tests**: Critical flows (Auth, Request).
